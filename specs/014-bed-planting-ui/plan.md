# Implementation Plan: Improved Bed Planting UI

**Branch**: `014-bed-planting-ui` | **Date**: 2026-03-07 | **Spec**: `specs/014-bed-planting-ui/spec.md`

---

## Summary

Replace the modal-based `PlantPicker` on the bed detail page with an always-visible inline plant panel. Introduce stamp mode (select plant → click cells to apply). Add a `DELETE /beds/:id/cells` backend endpoint for bulk clear. Frontend-dominant change; one new backend route.

---

**Language/Version**: Node.js 22 (backend) / React 19 + JSX (frontend)
**Primary Dependencies**: Express 5, Mongoose 9, TanStack React Query 5, React Router 7, Tailwind CSS 3, Lucide React (all already installed — no new packages)
**Storage**: MongoDB 7 — no new collections; additive write path on existing `GardenBed.cells`
**Testing**: Jest 29 + Supertest (backend unit), Playwright 1.x (E2E)
**Target Platform**: Web (desktop + mobile responsive)
**Project Type**: Web application (backend/ + frontend/ split)
**Performance Goals**: Clear bed completes in a single HTTP request regardless of cell count
**Constraints**: No new npm packages; must not break existing Playwright suite
**Scale/Scope**: Single page refactor + one new endpoint

## Implementation Approach

### Backend: `DELETE /beds/:id/cells`

```js
router.delete('/:id/cells', requireAccess('owner'), async (req, res) => {
  const bed = await GardenBed.findById(req.params.id);
  if (!bed) return res.status(404).json({ error: 'Bed not found' });
  // requireAccess already validates ownership via ownerId
  bed.cells = [];
  await bed.save();
  res.json(bed);
});
```

Response shape identical to `GET /beds/:id`. No new fields, no breaking changes.

### Frontend: `BedDetail.jsx` refactor

**Layout change**: Two-column flex on desktop (`lg:flex-row`), stacked on mobile (`flex-col`).

```
┌─────────────────────────────┬──────────────────────┐
│  Bed grid                   │  Plant panel         │
│  (scrollable if large)      │  [ Search...       ] │
│                             │  ┌──────────────────┐ │
│                             │  │ 🍅 Tomato        │ │
│                             │  │ 🥕 Carrot        │ │
│                             │  │ 🥦 Broccoli      │ │
│                             │  └──────────────────┘ │
│                             │  [Clear bed]         │
└─────────────────────────────┴──────────────────────┘
```

**State**:
- `selectedPlant` — `Plant | null` — the active stamp plant
- `search` — `string` — search input value

**Cell click logic**:
```
if (!selectedPlant) → no-op
else if cell.plant === selectedPlant → clear cell (PUT plantId: null)
else → stamp cell (PUT plantId: selectedPlant._id)
```

**Mutation**: new `useClearBed` mutation calling `DELETE /beds/:id/cells`, invalidates `['beds', id]`.

**Remove**: `PlantPicker` component and `selectedCell` state — replaced entirely.
